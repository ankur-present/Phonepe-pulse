# PhonePe Pulse Data Visualization and Exploration

## Project Overview

This project presents an interactive Power BI dashboard developed using the PhonePe Pulse dataset. The dashboard provides insights into digital payment trends, transaction patterns, user growth, and geographical distribution of transactions across India.

## Objectives

* Analyze transaction trends over time.
* Identify top-performing states and districts.
* Visualize transaction distribution across India using geo maps.
* Track user growth and engagement.
* Create an interactive and user-friendly dashboard.

## Tools & Technologies Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Dashboard Features

### Executive Dashboard

* Total Transaction Amount
* Total Transactions
* Total Users
* Average Transaction Value
* YoY Growth %
* Year-wise Transaction Trend
* Transaction Type Analysis

### Geo Analysis

* State-wise transaction distribution across India
* Interactive map visualization

### District Performance Analysis

* Top 10 Districts by Transaction Amount
* Top 10 Districts by Transaction Count
* Top States by Registered Users

## Key Insights

* Digital transactions have shown significant growth over the years.
* Peer-to-peer payments account for the majority of transactions.
* Telangana and Maharashtra emerged as top-performing states.
* Bengaluru Urban and Hyderabad are among the top-performing districts.
